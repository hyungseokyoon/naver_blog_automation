# 콘텐츠 생성 스킬 (Content Generation)

이 스킬은 URL이나 키워드와 같은 원시 입력을 형식이 지정된 블로그 콘텐츠로 변환하는 데 중점을 둡니다.

## 주요 기능
- **Markdown 변환**: 마크다운 형식을 네이버 스마트에디터 ONE과 호환되는 블록 구조로 변환합니다.
- **콘텐츠 요약**: 블로그 작성을 위해 외부 웹페이지 콘텐츠를 분석하고 요약합니다.
- **톤 앤 매너 관리**: 블로그 주제에 맞춰 적절한 문체와 스타일을 적용합니다.

## 워크플로우
1. 입력 데이터(URL, 키워드 등)를 수신합니다.
2. 생성형 AI를 사용하여 초안 콘텐츠를 작성합니다.
3. 출력 형식을 [제목, [본문 블록 1, 본문 블록 2, ...], 태그] 구조로 구성합니다.
