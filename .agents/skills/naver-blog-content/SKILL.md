# 네이버 블로그 콘텐츠 생성 스킬 (Naver Blog Content)

이 스킬은 네이버 블로그만의 '이미지-텍스트-이미지' 블록 구조와 검색 최적화(C-Rank/D.I.A)를 고려하여 내용을 구성합니다.

## 주요 기능
- **시각적 분석**: 폴더 내 이미지들을 분석하여 각 사진에 어울리는 설명과 맥락을 생성합니다.
- **네이버형 블록 배치**: 이미지 한 장당 2~3줄의 텍스트가 따라오는 한국식 블로그 흐름을 생성합니다.
- **키워드 최적화**: 제목과 본문 초반에 주요 키워드를 자연스럽게 녹여내어 검색 가시성을 높입니다.
- **톤 앤 매너**: 친절하고 정보 전달력이 높은 '~해요', '~입니다' 중심의 구어체를 사용합니다.

## 생성 데이터 구조 (JSON)
AI는 다음 형식을 준수하여 `post.json`을 생성해야 합니다:
```json
{
  "title": "블로그 제목",
  "content": [
    {"type": "text", "value": "첫 머리 문구"},
    {"type": "image", "file": "photo1.jpg", "caption": "사진 설명"},
    {"type": "text", "value": "사진에 대한 상세 설명 및 팁"},
    {"type": "image", "file": "photo2.jpg", "caption": "두 번째 사진 설명"}
  ],
  "tags": ["태그1", "태그2", "태그3"]
}
```
