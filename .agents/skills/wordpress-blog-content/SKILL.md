# 워드프레스 블로그 콘텐츠 생성 스킬 (WordPress Blog Content)

이 스킬은 워드프레스의 마크다운 기반 에디터 구조와 SEO 플러그인(Yoast, RankMath 등)을 고려하여 내용을 구성합니다.

## 주요 기능 (준비 중)
- **Gutenberg 블록 최적화**: 마크다운을 워드프레스 전용 블록 태그로 변환합니다.
- **특성 이미지(Featured Image) 설정**: 포스팅의 대표 이미지를 선정하고 메타데이터를 작성합니다.
- **카테고리 및 태그 관리**: 워드프레스 계층형 카테고리에 맞는 분류를 제안합니다.

## 생성 데이터 구조 (JSON 예정)
- 네이버와 유사한 구조를 갖되, HTML/Markdown 출력을 포함할 예정입니다.
